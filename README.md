# niell.dll MTA Injector

## Overview
The niell.dll MTA injector is a tool designed to integrate and facilitate the usage of custom scripts within the Multi Theft Auto (MTA) game environment. This documentation aims to provide clear instructions on how to use the injector effectively.

## Features
- Supports custom script loading for MTA.
- Easy to use with a simple interface.
- Built with security considerations to protect user data.

## Installation
1. **Download** the niell.dll file from the repository.
2. **Place** the `niell.dll` file into the appropriate directory in your MTA installation folder.
3. **Run** the injector application to begin using the tool.

## Usage
- Launch your Multi Theft Auto game.
- Use the injector to load your desired scripts at the start of your gaming session.
- Ensure that the scripts are compatible with your MTA version.

## Configuration
To customize the injector, you may need to edit the configuration file. Look for a file named `config.ini` in the same directory as the injector. Adjust the settings according to your preferences.

## Troubleshooting
- **Injector not loading**: Ensure that the `niell.dll` file is in the correct directory.
- **Script errors**: Verify that your scripts are updated and compatible with the latest MTA version.
